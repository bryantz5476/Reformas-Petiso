Eres un Desarrollador Frontend Experto en Alta Conversión y animaciones fluidas. Tu misión es construir una Single Page Application (SPA) para una empresa de reformas de interiores y arquitectura de lujo. 

ESTRICTAMENTE PROHIBIDO: 
- Crear cualquier tipo de Backend o usar ExpressJS.
- Usar el color amarillo. 
- Usar transiciones simples de opacidad (fade-ins aburridos). Quiero ver físicas, escalas, rotaciones y gradientes vivos. La web debe sentirse líquida e impactar en los primeros 3 segundos.

STACK TÉCNICO OBLIGATORIO: React, Vite, Tailwind CSS y lucide-react para los iconos. Asegúrate de que las versiones sean 100% compatibles.

PALETA DE COLORES: Tonos modernos, elegantes y de confianza. Usa combinaciones de Slate (Gris Pizarra), Deep Navy (Azul Marino Oscuro) y acentos en Blanco Puro y Plata. Cero colores ambiguos.

COREOGRAFÍA VISUAL Y ESTRUCTURA (PLAN 510€ - 7 SECCIONES):
Cada sección y elemento DEBE utilizar EXCLUSIVAMENTE los siguientes efectos de UI indicados. Implementa el código CSS/Tailwind necesario para que funcionen a la perfección:

1. SECCIÓN HERO:
- Fondo: Efecto "Fluid Glass" (Efecto vidrio esmerilado con orbes de luz en tonos fríos moviéndose lentamente por detrás).
- Título principal: Efecto "Split Text" (Las letras aparecen una por una con stagger).
- CTA (Botón principal): Efecto "Glare Hover".

2. SECCIÓN BANNER INFINITO (Partners/Marcas de materiales):
- Contenedor: Banner con movimiento infinito horizontal para mostrar los logos de los materiales de alta calidad que usa la empresa.

3. SECCIÓN DE SERVICIOS (Cocinas, Baños, Integrales):
- Tarjetas de servicio: Efecto "Spotlight Card" (Al pasar el mouse, un foco de luz sigue el cursor con un efecto radial gradient). 
- Textos destacados de los servicios: Efecto "Gradient Text" (El texto tiene un gradiente animado que fluye en tonos plata/azules).

4. SECCIÓN GALERÍA DE PROYECTOS (El Portafolio):
- Layout: Efecto "Magic Bento" (Grid irregular estilo Apple, con celdas de distintos tamaños mostrando los mejores proyectos).
- Imágenes dentro del grid: Al hacer hover, la imagen debe tener físicas ligeras de escala.

5. SECCIÓN "EL ANTES Y DESPUÉS" (Imágenes Grandes):
- Animación de imagen: Efecto "Gradual Blur" (La imagen del proyecto terminado se enfoca progresivamente por zonas a medida que se hace scroll).

6. SECCIÓN DE RESEÑAS:
- Layout: Un "Slider" para mostrar las opiniones de los clientes. 
- Tarjetas del slider: Deben sentirse fluidas al arrastrar o cambiar.

7. SECCIÓN DE CONTACTO / PRESUPUESTO (CRÍTICO PARA CONVERSIÓN):
- Formulario de contacto integrado para solicitar presupuestos.
- Contenedor del formulario: Efecto "Tilted Card" (Efecto 3D donde la tarjeta del formulario se inclina sutilmente hacia el mouse dando perspectiva).
- Botón de envío: Efecto "Glare Hover".
